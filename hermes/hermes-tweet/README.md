# Hermes Tweet Skill

![Skill](https://img.shields.io/badge/skill-hermes__tweet-blue)
![Version](https://img.shields.io/badge/version-1.0.0-green)
![License](https://img.shields.io/badge/license-MIT-brightgreen)

Use the native [Hermes Tweet](https://github.com/Xquik-dev/hermes-tweet)
plugin for X/Twitter automation from Hermes Agent.

## Features

- Search tweets and summarize X conversations
- Read tweet replies and public tweet context
- Look up users, account status, and trends
- Monitor tweets, keywords, and social signals
- Prepare post tweets, post replies, DMs, follows, monitors, and webhooks with
  human approval gates
- Keep write-like tools disabled by default

## Quick Start

```bash
hermes plugins install Xquik-dev/hermes-tweet --enable
export XQUIK_API_KEY="xq_..."
export HERMES_TWEET_ENABLE_ACTIONS="false"
hermes tools list
```
